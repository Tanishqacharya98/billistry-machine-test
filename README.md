# Billistry Billing & Inventory Management System

## Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features

### Authentication
- Signup
- Login
- JWT Authentication
- Protected Routes

### Customer Management
- Add Customer
- View Customers
- Delete Customer
- Search Customer

### Product Management
- Add Product
- View Products
- Delete Product
- Search Product
- Product Filter
- Pagination

### Invoice Management
- Create Invoice
- Invoice PDF Download

### Dashboard
- Total Customers
- Total Products
- Total Invoices
- Total Revenue
- Dashboard Charts

### Bonus Features
- Dark Mode
- Pagination
- Charts
- PDF Download

## Installation

### Backend

cd backend

npm install

npm run dev

### Frontend

cd frontend

npm install

npm start

## Environment Variables

Create .env inside backend

PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key
